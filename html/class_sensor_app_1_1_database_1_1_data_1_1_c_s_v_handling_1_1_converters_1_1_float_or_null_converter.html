Custom converter for handling text to float conversion from a CSV. This converter attempts to parse the text as a float. If parsing is successful, it returns the parsed float value. Otherwise, it returns null for invalid or non-numeric text.

Converts a string from a CSV file to a float or null. The method attempts to parse the provided text as a floating-point number. If parsing is successful, the parsed float is returned. If parsing fails (i.e., the text cannot be converted to a valid float), the method returns null.

Parameters
textThe text from the CSV that is to be converted, which may represent a floating-point number.
rowThe current row being processed from the CSV file.
memberMapDataThe mapping data for the member being populated with the converted value.
Returns
A float value if the string can be parsed as a valid number; otherwise, null.
